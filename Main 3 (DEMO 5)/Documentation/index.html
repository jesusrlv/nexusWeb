Introduction

Accent is a creative HTML Template, which includes a wide variety of elements. It has elegant and corporate design which will instantly impress visitors.

The Accent template comes with 4 super awesome backgrounds and 3 menu variants and supports professional HTML5 CSS3 technology. Great 11 demo versions and a lot of interactive elements allow to create stylistically specific website variants. It boasts all the features you need to build a business website. We keep Accent optimized, responsive and flexible for everyone.

Please do not forget to rate Accent on ThemeForest. This can be done in Downloads area. Just select Accent in the Download list and rate it. We really appreciate your feedback and will be grateful for your opinion.

Getting Started

This guide is targeted at people, who have at least basic knowledge of html markup language. Furthermore css and javascript knowledge is advised. If you do not feel yourself well with understanding these technologies you should consider finding someone to assist you.

This template can't be installed into any of available CMS (Wordpress, Drupal, etc.).

This documentation will demonstrate the custom elements used in template and help you with their customization.

Installation

Template installation is pretty easy. All required files are in every version so you can delete unused. Here are the required steps:

  1. Unzip the HTML folder
  2. Choose the desired variant
  3. Rename it to index.html
  4. Upload it through FTP

Files Structure

  • /css/ - CSS styles folder
  • /fonts/ - webfont icons folder
  • /images/ - folder for images
  • /js/ - JavaScript files folder
  • /media/ - Video Background files folder
  • /sass/ - Syntactically Awesome Stylesheets files folder
  • index.html - main HTML file

Our template supports different types of background.

  • Video Background
  • Slider Background

Video Background

Video background effect doesn't work on tablets and phones. For different web-browsers we need to upload different types of video: .mp4, .ogg, .webm are the needed ones. To switch on the video background, insert the following code under any section after the first tag that opens that section:

									
  1. <div class="background bg-video">
  2.    <div class="layer" style="background-image: url(images/slide/slide-6.jpg);"></div>
  3.    <video autoplay="autoplay" loop="loop" muted="muted"> 
  4.       <source src="media/cloud.mp4" type="video/mp4">
  5.       <source src="media/cloud.ogg" type="video/ogg">
  6.       <source src="media/cloud.webm" type="video/webm">
  7.     </video>
  8.     <div class="layer" style="background-color:rgba(205, 51, 51, 0.85);"></div>
  9. </div>
  • First "layer" - the picture for the tablets and phones
  • "muted" - you can delete this class and the video will be played with the sound

Retina

Retina Ready is a feature for displaying the high resolution images on the modern devices. It is created to show the true view of the HD images.

In our theme to enable the Retina for images you should have two images:

  • first one has the standard size and a standard name (slide-2.jpg)
  • second one must be twice bigger then the origin and have additional annex @2x (slide-2@2x.jpg

Both images must be in the same folder!

So, how we can enable it in our html template? You should insert a class replace-2x into the 'img' tag. Example:

  1. <img class="replace-2x" src="images/slide-2.jpg" alt="">

This is an example of CSS code that displays how we can connect our Retina image to the original image of the background:

  1. .part-2 {
  2.   background-image: url('../images/slider/slide-2.jpg');
  3. }
  4. @media (-webkit-min-device-pixel-ratio: 2) {
  5.   .part-2 { background-image: url('../images/slider/slide-2@2x.jpg'); 
  6.   background-size: cover; }
  7. }
